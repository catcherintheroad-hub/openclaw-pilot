# Security Policy

## Scope

This project influences agent planning and execution handoff. Security issues that affect command integrity, execution scope, prompt injection handling, runtime hygiene, or state leakage should be reported responsibly.

## Please report privately if the issue involves

- command injection,
- unsafe execution packet generation,
- state leakage across pilots,
- stale runtime confusion that could cause unintended execution,
- or filesystem path issues affecting runtime safety.

## What to include

- reproduction steps,
- environment details,
- whether the issue happens only in a contaminated runtime,
- relevant logs,
- and impact assessment.
