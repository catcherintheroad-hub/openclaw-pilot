# Command Pilot Examples

## Preview

`/pilot --preview 把 OCAX 首页和 roles 页统一成 linear 风格，先审计再改，不要动后端`

## Draft Only

`/pilot --draft 帮我把 OCAX 目录里没用的旧文件都清理掉`

## Run

`/pilot --run 为当前项目补一份安装说明和 smoke test，先审计现状再动手`

## Confirm Pending

`/pilot confirm <approval-id>`

## Discard Pending

`/pilot discard <approval-id>`
