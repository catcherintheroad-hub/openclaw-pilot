# Product Positioning

## One-sentence definition

OpenClaw Pilot compiles vague intent into executable AI work instructions.

## What problem it solves

Most users do not know how to phrase work so an agent can reliably continue it.

They have:
- an idea,
- a target outcome,
- a few constraints,
- but not a good agent-ready work packet.

Pilot converts that gap into a repeatable workflow.

## Who it is for

- OpenClaw users who know what they want but not how to prompt it well
- Operators who want clearer task handoff
- Builders who want stage-aware continuation rather than one-shot prompting

## Why this matters

Prompting is fragile. Compiled work packets are more durable.
